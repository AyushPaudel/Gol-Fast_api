Game of Thrones Real Estate Game API

A text-based game where Jon Snow ventures into modern-day real estate. Make choices to balance happiness and wealth!

How to Play

1. Start New Game

curl -X POST \
http://localhost:5000/api/game \
-H "Content-Type: application/json" \
-d '{}'

2. Make a Choice

curl -X POST \
http://localhost:5000/api/game \
-H "Content-Type: application/json" \
-d '{"choice": "1"}'

Replace "1" with your choice number (1-3)

3. Reset Game

curl -X POST \
http://localhost:5000/api/reset \
-H "Content-Type: application/json" \
-d '{}'

Game Rules

  • Starting stats: Happiness (30/100) and Wealth ($100,000)
  • Game runs for maximum 10 turns
  • Win condition: Happiness > 80 AND Wealth > $150,000
  • Lose if: Happiness reaches 0 OR Wealth reaches 0
